Instructions:
This is a helicopter
1. Download the archive uh1d_bin-main.zip
2. Run Dayz_tools https://store.steampowered.com/app/830640/DayZ_Tools/
3. Launch and select AddonBuilder in it
4. In the Addons source directory line, specify uh1d_bin-main \ HelicopterSIB_UH1D - the HelicopterSIB_UH1D folder
5. For those who decided to ignore in 4 point, I repeat, DO NOT PACK uh1d_bin-main the folder !!! find HelicopterSIB_UH1D a folder in it and pack it in pbo ...!!!
6. Uncheck Binarize
7. Pack it.
8. You should get HelicopterSIB_UH1D.pbo
9. Place it pbo in any of your mods with a link to the source https://github.com/moddayz




Инструкция:
Это вертолет
1. Скачиваете архив uh1d_bin-main.zip
2. Запускаете Dayz_tools https://store.steampowered.com/app/830640/DayZ_Tools/
3. Запускаете и выбираете в нем AddonBuilder
4. В строке Addons source directory указываете uh1d_bin-main \ HelicopterSIB_UH1D - папка HelicopterSIB_UH1D 
5. Уберите галочку Binarize
6. Запаковываете.
7. Должен получиться HelicopterSIB_UH1D.pbo
8. Размещаете pbo в любой свой мод с указанием ссылкой на источник https://github.com/moddayz
